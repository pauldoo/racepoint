RacePoint

Free software for managing your pigeon racing season.

About:

    This software is intented for pigeon racers and racing club
    organisers.

    Stores bird clock times for each member during the racing season.

    Calculates race results with bird velocities, and pool results.

    Created by Paul Richards <paul.richards@gmail.com> and released as
    free software under the ISC license.

    http://www.pauldoo.com/racepoint/


Instructions:

    RacePoint requires Java to be installed on your computer before it
    can run. You can get this here: http://www.java.com/getjava/

    To run RacePoint double click the "RacePoint.jar" file.
